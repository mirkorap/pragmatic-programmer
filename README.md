# Pragmatic Programmer

Questo book ha lo scopo di riassumere in breve tutti i concetti espressi all'interno del libro "The Pragmatic Programmer" di Andrew Hunt e David Thomas. Di seguito gli argomenti trattati all'interno del libro:

* [Una filosofia pragmatica](https://mirkorap16.gitbook.io/pragmatic-programmer/una-filosofia-pragmatica)
* [Un approccio pragmatico](https://mirkorap16.gitbook.io/pragmatic-programmer/un-approccio-pragmatico)
* [Gli strumenti di base](https://mirkorap16.gitbook.io/pragmatic-programmer/gli-strumenti-di-base)
* [Paranoia pragmatica](https://mirkorap16.gitbook.io/pragmatic-programmer/paranoia-pragmatica)
* [Piegarsi o spezzarsi](https://mirkorap16.gitbook.io/pragmatic-programmer/piegarsi-o-spezzarsi)
* [Mentre codificate](https://mirkorap16.gitbook.io/pragmatic-programmer/mentre-codificate)
* [Prima del progetto](https://mirkorap16.gitbook.io/pragmatic-programmer/prima-del-progetto)
* [Progetti pragmatici](https://mirkorap16.gitbook.io/pragmatic-programmer/progetti-pragmatici)
